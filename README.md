# End-to-End Machine Learning Deployment using Flask

## Description
This project demonstrates an end-to-end Machine Learning pipeline including model training, backend API development, and web deployment using Flask.

## Objectives
- Build a machine learning model
- Deploy model using Flask
- Create a user-friendly web interface
- Generate predictions from user inputs

## Project Pipeline

Dataset → Data Preprocessing → Model Training → Model Saving → Flask Integration → Web Prediction
